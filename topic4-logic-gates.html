Logic Gates & Boolean Algebra

Logic gates, truth tables, Boolean algebra laws, simplification, logic circuit diagrams, Karnaugh maps, and universal gates.

Paper 1 · AS & A Level ~12% of Paper 1 marks

✦ Learning Objectives

By the end of this topic you should be able to:

  • State the function and draw the symbol for AND, OR, NOT, NAND, NOR, XOR
  • Construct and complete truth tables for any logic expression
  • Write Boolean expressions using standard notation (·, +, overbar)
  • Apply all Boolean algebra laws including De Morgan's theorems
  • Simplify Boolean expressions algebraically, showing every law used
  • Draw a logic circuit from a Boolean expression
  • Derive a Boolean expression from a given logic circuit
  • Use a Karnaugh map (K-map) to find a minimal Sum of Products expression
  • Explain why NAND and NOR are universal gates and show equivalents
  • Describe the half adder and its gate implementation

1 Logic Gates

A logic gate is an electronic circuit with one or more binary inputs and a single binary output. The output is fully determined by its inputs according to a Boolean function.

AND

⊓

Output 1 only when all inputs are 1

OR

⊔

Output 1 when at least one input is 1

NOT

¬

Inverts the single input

NAND

⊼

AND then invert — universal gate

NOR

⊽

OR then invert — universal gate

XOR

⊕

Output 1 when inputs differ

Exam tip: The bubble (small circle) on the output means invert. NAND = AND + bubble; NOR = OR + bubble. If you forget a gate symbol, drawing the equivalent gates joined together also earns full marks.

3.1 Operator Precedence

Evaluate in this order (highest to lowest):

  1. Parentheses — innermost first
  2. NOT (Ā)
  3. AND (·)
  4. OR (+)
Example: A + B · C̄ means A + (B · (NOT C)) — AND binds before OR.

4 Boolean Algebra Laws

These identities let you simplify expressions without constructing a truth table. You must state the law name for each step in the exam.

4.1 Identity, Null & Idempotent

Identity
A + 0 = A  |  A · 1 = A
Null / Annihilator
A + 1 = 1  |  A · 0 = 0
Idempotent
A + A = A  |  A · A = A
Complement
A + Ā = 1  |  A · Ā = 0
Double Negation
Ā̄ = A

4.2 Commutative, Associative & Distributive

Commutative
A + B = B + A  |  A · B = B · A
Associative
(A+B)+C = A+(B+C)
(AB)C = A(BC)
Distributive
A(B+C) = AB + AC
A+BC = (A+B)(A+C)

4.3 De Morgan's Theorems ★

De Morgan 1
(A · B)' = Ā + B̄
NAND = OR of complements
De Morgan 2
(A + B)' = Ā · B̄
NOR = AND of complements
Memory trick: Break the bar, change the operator. (A·B)' → Ā + B̄ (dot becomes plus). (A+B)' → Ā · B̄ (plus becomes dot).

4.4 Absorption Laws

Absorption 1
A + AB = A
Absorption 2
A(A + B) = A
Useful identity
A + ĀB = A + B

5 Simplifying Boolean Expressions

Simplifying reduces the number of gates needed in a circuit, lowering cost, power consumption, and propagation delay.

5.1 Worked Example 1

Simplify F = AB + AB̄

Step-by-step
F = AB + AB̄Given
= A(B + B̄)Distributive — factor A
= A · 1Complement law: B + B̄ = 1
= AIdentity law: A · 1 = A
Result: The circuit needs only a wire — no gates at all!

5.2 Worked Example 2

Simplify F = ABC + ABC̄ + AB̄C

Step-by-step
= AB(C + C̄) + AB̄CFactor AB from first two terms
= AB · 1 + AB̄CComplement law: C + C̄ = 1
= AB + AB̄CIdentity: AB · 1 = AB
= A(B + B̄C)Factor A
= A(B + C)Absorption: B + B̄C = B + C

5.3 Worked Example 3 — De Morgan's

Simplify F = (A + B)' · C + A · C

Step-by-step
= (Ā · B̄) · C + ACDe Morgan's theorem on (A+B)'
= ĀB̄C + ACRemove redundant brackets
= C(ĀB̄ + A)Factor C
= C(A + B̄)Absorption: ĀB̄ + A = A + B̄
Exam strategy: Show every step with the law name in brackets. Even if your final answer is wrong, you still score marks for correct intermediate steps.

6 Logic Circuit Diagrams

A logic circuit is a diagram connecting gate symbols with wires. You need to:

  • Draw a circuit from a Boolean expression
  • Derive a Boolean expression from a circuit
  • Trace a circuit to produce its truth table

6.1 Draw from Expression

Expression: Q = (A · B) + (Ā · C)

Circuit for Q = (A · B) + (Ā · C) — the filled dot on the A wire is a junction point

Junction dots: A filled dot (•) on crossing wires means they are connected. No dot means the wires cross without connecting. Always draw dots at branch points.

6.2 Reading an Expression from a Circuit

  1. Label each gate's output with its Boolean function (e.g. G1 = A · B)
  2. At the final output gate, write the full expression using the labelled sub-expressions
  3. Simplify if asked

6.3 Half Adder

A half adder adds two 1-bit numbers, producing a Sum and a Carry output. It is a key application of logic gates in arithmetic circuits.

Half Adder
ABSum (S)Carry (C)
0000
0110
1010
1101

S = A ⊕ B (XOR)    C = A · B (AND)

Two half adders joined with an OR gate on the carries form a full adder, which handles a carry-in from a previous stage and enables multi-bit addition.

7 Karnaugh Maps (K-maps)

A K-map is a visual tool for simplifying Boolean expressions. Adjacent cells differ by exactly one variable (Grey code ordering). Grouping 1s in powers of 2 eliminates variables.

Why K-maps? Algebraic simplification can miss the minimum form. K-maps provide a systematic method that always finds the minimal Sum of Products (SOP) expression.

7.1 2-Variable K-map

B=0B=1
A=0m₀m₁
A=1m₂m₃

7.2 3-Variable K-map

A \ BC00011110
0m₀m₁m₃m₂
1m₄m₅m₇m₆

Columns go 00→01→11→10 (Grey code). Columns 10 and 00 are also adjacent — the map wraps around.

7.4 K-map Rules

Group size
Must be a power of 2: 1, 2, 4, 8…
Shape
Groups must be rectangular — no diagonals
Wrap-around
Edges are adjacent — the map wraps
Largest first
Always make groups as large as possible
Overlap
Groups can overlap; a 1 may be in multiple groups
Don't-cares
X can be 0 or 1 — use them to enlarge groups

8 NAND & NOR as Universal Gates

Any Boolean function can be constructed using only NAND gates, or only NOR gates. This is valuable in chip manufacturing — a single gate type simplifies the production process.

8.1 Building AND, OR, NOT from NAND

Gate neededNAND equivalent
NOT ANAND(A, A)
A AND BNAND( NAND(A,B), NAND(A,B) )
A OR BNAND( NAND(A,A), NAND(B,B) )

8.2 Building AND, OR, NOT from NOR

Gate neededNOR equivalent
NOT ANOR(A, A)
A OR BNOR( NOR(A,B), NOR(A,B) )
A AND BNOR( NOR(A,A), NOR(B,B) )
Proof that NAND gives NOT: NAND(A,A) = (A·A)' = A' (idempotent then complement). One gate, with both inputs tied together, acts as a NOT.

Question 2 · 5 marks

(a) Simplify F = ĀBC + ABC + AB̄C using Boolean algebra. [3]

(b) State one advantage of simplifying before building a circuit. [1]

(c) State the law used when A + Ā = 1. [1]

(a)

F = ĀBC + ABC + AB̄C

= BC(Ā + A) + AB̄C  [factor BC]

= BC · 1 + AB̄C  [complement law]

= BC + AB̄C  [identity]

= C(B + AB̄)  [factor C]

= C(B + A)  [absorption: B + AB̄ = B + A]

F = C(A + B)

Award 1 mark each: correct factoring; correct complement law; correct final form.

(b) Any ONE of: fewer gates / lower cost / lower power / less propagation delay / smaller circuit area

Question 3 · 6 marks

The Boolean expression for a logic circuit is: F = (A + B) · (Ā + C)

(a) Draw the logic circuit. [3]

(b) Expand and simplify using De Morgan's theorem and Boolean algebra. [3]

(a) Circuit features (1 mark each):

  • NOT gate with input A producing Ā
  • OR gate 1: inputs A and B; OR gate 2: inputs Ā and C
  • AND gate combining both OR outputs to give F, with correct connections

(b)

F = (A + B)(Ā + C)

= AĀ + AC + ĀB + BC  [distributive — expand]

= 0 + AC + ĀB + BC  [complement: AĀ = 0]

= AC + ĀB + BC

Using consensus theorem: BC is redundant (covered by AC and ĀB):

F = AC + ĀB  [1 expand, 1 eliminate AĀ, 1 correct final form]

Key Boolean Laws

  • A·0=0, A+1=1
  • A·1=A, A+0=A
  • A·Ā=0, A+Ā=1
  • Ā̄=A (double negation)
  • De M. 1: (AB)'=Ā+B̄
  • De M. 2: (A+B)'=ĀB̄
  • A+AB=A (absorption)
  • A+ĀB=A+B

Common Mistakes

  • Not stating law names when simplifying — loses marks
  • Wrong Grey code column order in K-map: must be 00,01,11,10
  • Diagonal groups in K-maps — invalid, must be rectangular
  • Missing junction dots on circuit diagrams
  • Breaking De Morgan's bar partially — must break the whole bar at once
